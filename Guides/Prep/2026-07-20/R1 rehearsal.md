### 読み替え（16:00–16:20）の手順

1. 仮テーマ原文を Cursor チャットに貼る
2. プロンプト例:

```
このハッカソンテーマを字面どおり解釈しない「読み替え案」を5つ。
審査員が驚く尖りを含めて。各案はタイトル1行＋説明2行。
temperature は高めで。
```

3. 5案から **1案** を選ぶ（尖り = 画像生成 or 画像入力のどちらか **当日用に1つ** だけ決める）
4. 詳細: [[Concepts/model selection|model selection]]

### 流し込み（16:20–17:20）の手順

1. 選んだ読み替えを `app/page.tsx` の見出し or system プロンプトに **1–2行** 反映
2. Agent が動くことを確認（tool ログが見えること）
3. 尖りは **触らなくてよい**（火〜水で実装）
4. push → Vercel 再デプロイ（URL がある場合）

3. 30秒: 「AI がここで tool を呼んだ」と **画面上のログ** を指す
