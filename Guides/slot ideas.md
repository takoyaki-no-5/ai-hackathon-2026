### F とは（セクシーカウンター型）

- **完成した実用ツール**より、「触った瞬間ヤバい／この先が面白そう」の **チャレンジ1発**
- 審査の「ハッカー魂」寄り。1st 3位の評価理由がまさにこれ
- 中身はだいたい **多モーダル1往復**: 画像/動画/音声 → スコアや判定 → 一言コメント

#### 向くとき / 向かないとき

- 向く: テーマが遊び・評価・判定・嘘・でたらめ等と自然につながる
- 向かない: 真面目な業務テーマなのに無理にギャグ判定を載せる／デモが許可ダイアログで詰まる
- 予選本命にはしにくい。**A/B が安定してから余力で乗せる**か、テーマがドンピシャのときだけ主役

#### 3分デモの型

1. 審査員に撮らせる／貼らせる（自分で事前デモ素材も用意）
2. 1〜2秒で結果が出る
3. 「なぜその点か」をAIが1行で言う
4. （余裕）改善アドバイスまで → 実質 A 寄せで強くできる

### いま押さえる最新レイヤ（7月時点の感覚）

- **Realtime / Live**: 声で即応答（OpenAI Realtime、Gemini Flash Live）→ デモ映えするが会場許可が鬼門
- **Vision**: 写真・スクショ診断は予選コスパ最強
- **Computer use**: ハッカー感は強いが予選3hの主役には不向き（Dの予備）
- **Multi-agent / 高機能 Responses**: 「AI活用が見える」演出向き
- **アクセス制限のある最新旗艦**（例: 一部プレビューモデル）に賭けない。**手に入る安定API**を本線に

### 写真一択は危険（被る）

Vision診断はコスパ良いが、**多くのチームが同じ穴に落ちる**。差は「何を入力にするか」。

#### 写真以外の入力・機能（広めに拾った）

| 系統 | できること（例） | 技術の手がかり | 通し |
|------|------------------|----------------|------|
| **声だけ** | 話し方・間・嘘っぽさ・幹事ボット | Realtime / Gemini Live / ElevenLabs | ○ |
| **ブラウザ内AI（オフライン寄り）** | 要約・翻訳・校正を **APIキーなし** でChrome内実行 | Chrome Built-in AI（Translator / Summarizer / Prompt） | ◎〜○ |
| **地図・場所** | 「今ここから徒歩10分でできること」を会話で | Mapbox MCP / Directions / Isochrone | ◎ |
| **手の動き** | 手で音を出す・ジェスチャで操作・指揮 | MediaPipe Hands + Web Audio / Tone.js | ○ |
| **MIDI・楽器** | 弾いた音→AIがコーチ | Web MIDI + LLM | ○（機材要） |
| **Webマイクの環境音** | 会場のざわめき→雰囲気ラベル（撮らない） | 音声分類 + LLM | ◎ |
| **スクショではなく「URLを渡す」** | 公開ページを取って要約・採点（審査員PCでも再現しやすい） | fetch + LLM / 軽量browser fetch | ◎ |
| **コード実行つき解答** | 問題→Python実行→根拠付き答え | Anthropic/OpenAI code execution | ◎ |
| **検索ログが見える調査** | 質問→検索・計算の途中が流れる | web search tool + UI | ◎ |
| **声のクローン／キャラ声** | テーマのキャラが喋る | ElevenLabs | ○ |
| **その場で曲** | テーマ曲を生成してBGMに | Suno系 / Eleven music 等 | ○△ |
| **3Dの世界を生成** | テキスト→歩き回れる空間 | World Labs Marble World API | △（待ち時間） |
| **エージェントが地図を動かす** | 会話でピン・ルートが変わる | Mapbox Location Agent / MCP | ○ |
| **Clipboard／選択テキスト** | 貼り付けた長文をその場で書き換え・論破 | Writer/Rewriter or LLM | ◎ |
| **端末センサー** | 傾き・位置（許可）×AIコメント | DeviceOrientation / Geolocation | ◎ |
| **二画面連携** | PCが本機、スマホがカメラ／コントローラ | QRで部屋コード（ハッカソン常連パターン） | ○ |
| **自前GPU** | 同じ問題をAPI vs 自宅モデル比較 | トンネル + フォールバック | △ |

#### 「写真チーム」に勝つ見せ方

- 入力が写真でも、**一連の体験**（診断→提案→やり直し）まで行く
- 写真を使わないなら、**審査員が3分で「他と違う入力」をした実感**を残す（声・地図・手・URL・ブラウザ内AI）
