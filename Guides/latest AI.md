### 最新おもしろAI（〜2026/7中旬）— 何ができるか

ラベル: **予選向き◎** / **デモ映え○** / **余裕・決勝△** / **開発で使う★**

---

### 頭がいいAI（文章・コード・考える）

| 名前 | 何ができるか |
|------|----------------|
| **GPT-5.6**（強い/普通/安いの3段階） | コードを書く・設計する・難しい調べ物。強いほど賢いが遅い・高い |
| **ツールをプログラムでつなぐ**（GPTの新機能） | 「検索して→計算して→まとめる」をAIが自分でつなぐ。待ちが減る |
| **複数AIを同時に動かす**（multi-agent） | 調べ役・書き役などを並行させて、最後に1つにまとめる |
| **Claude Fable 5** | いま一番強い部類のClaude。一部の危ない話題は弱いモデルに切り替わる |
| **Claude Sonnet 5 / Opus 4.8** | 普通に賢い相棒。コードや長文に強い。毎日使いやすい |
| **Gemini 3.1** | 文章だけでなく、写真・音声・動画もまとめて理解しやすい |
| **Muse Spark 1.1**（Meta） | 計画を立てて、道具を使い、画面操作やコーディングまでやるタイプ |
| **Grok 4.5** | コードやエージェント向き。比較的安めで開発ツールにも入り始めている |

---

### 声で話すAI

| 名前 | 何ができるか |
|------|----------------|
| **Realtime系（OpenAI）** | 電話みたいにリアルタイムで会話。割り込みや雑音にも少し強くなった |
| **GPT-Live** | 話しながら、裏で賢いモデルに難しい質問を投げて答えを持ってくる |
| **Gemini Flash Live** | 声で話している最中に、カメラ映像も見て返事できる。外国語にも強い |
| **xAIの音声エージェント作り** | 難しい設定なしで「電話応対AI」を組み立てられる |

※会場ではマイク許可が面倒。テキストでも動く逃げ道を用意する。

---

### 画像・動画を作る／見るAI

| 名前 | 何ができるか |
|------|----------------|
| **写真・画面を読ませる（Vision）** | 自撮り・スライド・スクショを見て採点・コメント・直し方を出す |
| **GPT Image など** | きれいなイラストやUIモックをすぐ生成・直し |
| **Sora 2** | テキストや画像から短い動画を作る（長い・きれいめも増えた） |
| **Veo 3.1**（Google） | 動画＋音まで一緒に作る。縦動画や「この写真たちから動画」も |
| **Muse Image**（Meta） | 画像を作るとき、自分で検索したりコードで図を描いたりして精度を上げる |
| **Muse Video / Runway / Kling** | 別会社の動画生成。長さ・画質・操作のしやすさで得意が違う |

---

### 自分で動くAI（エージェント）

| 名前 | 何ができるか |
|------|----------------|
| **Computer use** | 画面を見て、クリックや入力を人の代わりにやる |
| **MCP** | 「カレンダー」「検索」「DB」などをAIにプラグインみたいにつなぐ仕組み |
| **Claude Code のブラウザ** | コーディングAIがWebページを開いて読んで操作できる |
| **裏で走るエージェント** | 自分が別のことをしている間に、修正やPR作成まで進める |

---

### 開発で自分たちが使う道具

| 名前 | 何ができるか |
|------|----------------|
| **Cursor** | エディタ上でAIにコードを書かせる・直させる（今回の主武器） |
| **Claude Code / Codex / Grok Build** | ターミナルや専用画面で、もっと自律的にコード作業させる |
| **Agents / Responses API** | 上の「調べて・呼んで・まとめる」を自分のWeb作品に埋め込むためのAPI |

---

### 予選でやりすぎ注意

- 「いちばん新しいけど使えないかも」なモデルだけに頼る
- 画面自動操作を3時間のメインにする
- 審査の途中で長い動画生成を待つ
- 電話システムを当日初めて触る

---

### ハッカソンでの使い方（超要約）

1. **頭脳**: いつもの賢いモデル（文章・コード）
2. **一芸**: 写真だけにしない（声・地図・手・ブラウザ内AI・地図・3D世界など → [[Guides/slot ideas|slot ideas]]）
3. **作り方**: Cursorで大量にコード生成
4. **おまけ**: 動画・きれい画像はオープニングやデモ用
