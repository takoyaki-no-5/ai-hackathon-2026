# Profile: F（初心者・見せる枝）

`secrets/operator.md` にコピーすると、エージェントは **Fが操作中** とみなす。

```yaml
id: F
name: F
role: 見せる枝
level: プログラミング初心者
```

## 知識（要約）

- プログラミング: **初心者**
- OS: **macOS**（コマンドは zsh / `cp`。PowerShell前提にしない）
- **Cursor**: K と同じ開発環境。画面確認は Chrome
- ハッカソンではデモ・文言・動作確認・素材が主
- 実装の本線は K。F は Cursor と一緒に **起動・確認・文言** まで。壊れそうなら触らない

## エージェントへの指示

- 返答の前提は常に **operator=F**
- 専門用語はかみくだく。長い前提講義はしない（必要な一文だけ）
- **F が困ったら、まずあなた（Cursor）が手順まで出す**。K に聞くのは初回URL・秘密・デプロイ・リハの役割・2回試しても直らないときだけ
- コード変更は最小・壊しにくいものに限る。デプロイやキー操作は K に任せるよう促す
- デモ台本・審査員への説明・UI文言・確認手順を優先して手伝う
- 日次は [[Study F/README|Study F]]（困ったら Cursor → ダメなら K）
- 詳細は `team.md` の **F** を正とする（追記されたらそれを読む）
