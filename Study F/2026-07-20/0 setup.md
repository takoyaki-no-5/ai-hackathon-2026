### 0. 準備

枠: **最初の5分**。Mac。

### 完了の定義

- [ ] Cursor が開いている
- [ ] **Chrome** が開いている（Brave ではない）
- [ ] ターミナルで `node -v` / `pnpm -v` に数字が出る

### 自分で開く URL

| 用途 | URL |
|------|-----|
| Chrome が無いとき | https://www.google.com/chrome/ |
| Node が無いとき（Cursor に相談してから） | https://nodejs.org/ |

### 手順

1. **Cursor** を開く
2. **Chrome** を開く（無ければ上の URL からインストール）
3. Cursor → 上部メニュー → **Terminal** → **New Terminal**
4. 次を打つ:

```bash
node -v    # 22 前後
pnpm -v    # 数字が出れば OK
```
