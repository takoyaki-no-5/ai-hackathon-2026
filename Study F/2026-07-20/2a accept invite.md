### 2a. GitHub 招待を Accept

枠: **5分**。**K が招待を送ったあと**、clone の前にやる。

### 完了の定義

- [ ] GitHub 通知 or メールで招待を見つけた
- [ ] **Accept invitation** を押した
- [ ] リポが開ける（下の URL）
- [ ] K に「Accept できた」と Discord 等で送った

### 自分で開く URL

| 用途 | URL |
|------|-----|
| 招待・通知 | https://github.com/notifications |
| リポ | https://github.com/takoyaki-no-5/ai-hackathon-2026 |

### 手順

1. Mac の Chrome で https://github.com/notifications を開く（GitHub にログイン `@fujiwara1406-cpu`）
2. `takoyaki-no-5/ai-hackathon-2026` の招待 → **Accept**
3. メールだけのときも同じ。**Accept** を押す
4. リポ URL を開いて表示できるか確認
5. K に送る:

```
Accept できた。
```

### Cursor に任せる

招待が見つからない・Accept が押せないとき:

```
GitHub のリポ Collaborator 招待を Accept したい。
アカウント: fujiwara1406-cpu
リポ: takoyaki-no-5/ai-hackathon-2026
Mac / Chrome。打つ操作と開く URL だけ教えて。
```

2回ダメなら K。

### 触らない

- `secrets/`・`.env`・APIキー
- リポの Settings・Collaborators 画面（K の作業）
