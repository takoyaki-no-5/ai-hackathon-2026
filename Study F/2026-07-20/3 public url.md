### 3. 公開 URL

枠: **K が作ってから**。作るのは K。F は開くだけ。

### 完了の定義

- [ ] K から `https://....vercel.app` が届いた（まだなら R1 は localhost で可）
- [ ] Mac の Chrome でその URL が開く
- [ ] できればスマホの Chrome でも開く

### 自分で開く URL

| 用途 | URL |
|------|-----|
| 公開 URL（K から届く） | `https://＜名前＞.vercel.app` |
| ローカル代替 | http://localhost:3000 |

### 手順

1. K から URL が届くのを待つ（Discord 等）
2. Mac Chrome のアドレス欄に貼って Enter
3. スマホ Chrome でも同じ URL を開く

まだ無ければこのタスクはスキップして [[Study F/2026-07-20/R1 rehearsal|R1]] へ。
